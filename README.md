# Leftover-food-to-poor-
A project to supply leftover food to the poor. Created by Krishna Sai Dondapati
Objective	Reduce food waste and hunger by providing a digital medium for donating surplus food.
📂 Tech Stack	Frontend: HTML/CSS/JS or React / Flutter<br>Backend: Node.js / PHP / Firebase<br>Database: MongoDB / MySQL / Firebase Firestore
🧑‍💻 Users (Roles)	1. Donor – Posts available food<br>2. Volunteer/NGO – Views and accepts requests<br>3. Admin – Manages users and approvals
🧩 Key Features	- Donor registration & food post<br>- Geo-location for nearby donors<br>- Volunteer/NGO food pickup request<br>- Notifications (SMS/Email)<br>- Delivery status tracking
🔄 Workflow	1. Donor posts surplus food<br>2. Location-based listing visible to volunteers<br>3. Volunteer accepts and picks up<br>4. Confirmation sent to both parties
📱 UI Pages / Modules	- Login/Register<br>- Dashboard for Donors<br>- NGO/Volunteer Food List<br>- Food Donation Form<br>- History/Status Page
💾 Database Tables / Collections	- Users (role, name, contact, location)<br>- Food_Posts (type, quantity, expiry, donor_id)<br>- Requests (food_id, volunteer_id, status)<br>- Feedback
🚀 Future Enhancements	- AI to predict food spoilage<br>- Route optimization for pickups<br>- Multilingual support<br>- NGO verification badge
🔐 Security & Privacy	- OTP/Email verification<br>- Basic data encryption<br>- Role-based access control
