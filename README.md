  <h1># Food Waste Management System</h1>
  <img src="https://images.pexels.com/photos/6646915/pexels-photo-6646915.jpeg" 
       alt="Food Donation Cover" width="600">

  <p>
    The basic concept of this project <b>Food Waste Management</b> is to collect 
    the excess/leftover food from donors such as hotels, restaurants, marriage halls, etc., 
    and distribute it to needy people.
  </p>

  <h2>Tools and Technologies</h2>
  <ul>
    <li>Frontend: HTML, CSS, JavaScript</li>
    <li>Backend: PHP</li>
    <li>Webserver: XAMPP Server</li>
    <li>Database: MySQL</li>
  </ul>

  <h2>The System Has Three Modules</h2>
  <ul>
    <li>User</li>
    <li>Admin</li>
    <li>Delivery</li>
  </ul>

  <h3>User Module</h3>
  <img src="https://images.pexels.com/photos/6646931/pexels-photo-6646931.jpeg" 
       alt="User Donating Food" width="500">
  <p>
    The User module is designed for people who wish to donate their excess or leftover food 
    to help reduce food wastage. Users can register, login, and donate food. The system will 
    match their donation with the nearest needy people or organizations.
  </p>

  <h3>Admin Module</h3>
  <img src="https://images.pexels.com/photos/1181406/pexels-photo-1181406.jpeg" 
       alt="Admin Managing Donations" width="500">
  <p>
    The Admin module is for trusts, NGOs, and charities registered on the platform. 
    Admins can manage donations, track requests, and assign them to NGOs or charities.
  </p>

  <h3>Delivery Module</h3>
  <img src="https://images.pexels.com/photos/4393665/pexels-photo-4393665.jpeg" 
       alt="Delivery Person" width="500">
  <p>
    The Delivery Person module is for individuals providing pickup and delivery services. 
    They can view pickup and drop-off locations of food donations and ensure timely delivery.
  </p>

  <h2>How to Run</h2>
  <ol>
    <li>Download the project zip file</li>
    <li>Extract the file and copy the folder</li>
    <li>Paste it inside the root directory (for XAMPP: xampp/htdocs, for WAMP: wamp/www, for LAMP: var/www/html)</li>
    <li>Open PHPMyAdmin (<a href="http://localhost/phpmyadmin">http://localhost/phpmyadmin</a>)</li>
    <li>Create a database</li>
    <li>Import <b>demo.sql</b> file (inside the database folder)</li>
    <li>Run the script at <b>http://localhost/folderName</b></li>
  </ol>
